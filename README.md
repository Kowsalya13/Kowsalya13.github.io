# Kowsalya13.github.io

Personal portfolio — live at **[kowsalya13.github.io](https://kowsalya13.github.io)**

## Stack

Pure HTML · CSS · Vanilla JS — no frameworks, no build step.

## Features

- Terminal boot sequence on load (`> ./whoami` → `content fetched. ✅`)
- Three-column layout: sidebar nav · main content · live dashboard
- Scroll-triggered fade-in for each section
- Fully responsive (collapses gracefully to single column on mobile)
- Dark terminal aesthetic with green glow accents

## Sections

| Route | Content |
|---|---|
| `#home` | Intro, role, tag line |
| `#about` | Background summary |
| `#skills` | Tech stack grid |
| `#experience` | Domain expertise |
| `#projects` | Jenkins on EKS · GitOps Automation |
| `#certifications` | AWS SAA · CCP · Cisco · Coursera |
| `#connect` | GitHub · LinkedIn |

## Run locally

Open `index.html` in a browser — or use any static server:

```bash
npx serve .
# or VS Code Live Server
```
